Detalles

  • Console App inspirada en la WebApp "Spotify",adoptando la mayoria de sus funcionalidades.
  • Desarrollada mediante programacion estructurada(no POO), donde el fuerte se centra en una logica pura y simplista a la hora de manipular datos directamente desde la memoria mediante el uso de estructuras y estructuras compuestas aplicando punteros.
  • La misma permite el registro y login en la app para la gestion de playlist(limitada segun su plan). Ademas,en rol Administrador se realiza un CRUD completo sobre usuarios, canciones, albumes, artistas y playlists.
  • Se predispone el Clean Code y las buenas practicas basicas de programacion, asi como tambien las bases para la escalabilidad de proyecto.
  • Llevo un tiempo de 1 mes de desarrollo, aplicando modalidad Pair Programming presencial y Trello para realizar un diario de trabajo; asi como tambien github como control de versionado.Concluyendo en una presentacion evaluativa y expositiva.
